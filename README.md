# VetClinic - Veterinary Management System

VetClinic is a web application built with Ruby on Rails to manage the daily operations of a veterinary clinic.

## Key Features
* **Patient & Owner Management:** Complete registration of pets (species, breed, medical history) and their owners.
* **Medical Staff:** Administration of veterinarians and their specialties.
* **Appointment Control:** Medical appointment scheduling system tracking the lifecycle status (scheduled, in progress, completed, cancelled).
* **Treatment History:** Detailed record of treatments, medications, dosages, and clinical notes nested directly within each appointment.

## Technologies Used
* Ruby on Rails
* Relational Database (SQLite / PostgreSQL)
* Bootstrap (for styling and UI)
